# New Product Import Strategy

1. **Entity Model Setup**: Önce entity model / brands / bottlers kurulacak.
2. **Staging Table**: Sonra product candidate staging tablosu açılacak.
3. **Approval Flow**: Sonra manual review UI veya CSV approval yapılacak.
4. **Import Products**: Approved_new_products ayrı import edilecek.
5. **Attach Notes**: Tasting notes product importtan sonra bağlanacak.
6. **Isolation**: Hiçbir yeni ürün doğrudan whiskies master’a yazılmayacak.
